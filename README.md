# Metal Gear Rising: Revengeance RedTrainer
The trainer was based on [ImGUI](https://github.com/ocornut/imgui), [libmem](https://github.com/rdbo/libmem) 2.14, and [DX9-BaseHook](https://github.com/rdbo/DX9-BaseHook)

## Features
### Stats
- Change health
- Change money
- Invincibility
- Infinite energy
### Customization
- Change character
- Change body
- Change weapon
- Change wig
### Movement
- Change game speed
- Fly mode
